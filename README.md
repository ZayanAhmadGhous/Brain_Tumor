# 🧠 Brain Tumor Detection using CNN

A deep learning-based web application that detects brain tumors from MRI images using a Convolutional Neural Network (CNN). The model is deployed on Hugging Face for real-time predictions.

---

## 🚀 Project Overview

This project uses Deep Learning and Computer Vision to classify brain MRI images into:

* Tumor Detected
* No Tumor Detected

It aims to provide a fast and automated tool for preliminary screening.

---

## 🛠️ Tech Stack

* Python
* TensorFlow / Keras
* Convolutional Neural Networks (CNN)
* NumPy & Pandas
* Matplotlib / OpenCV
* Hugging Face Spaces
* Flask / Gradio

---

## 🧠 Model Architecture

* Convolutional Layers
* Max Pooling Layers
* Flatten Layer
* Dense Layers
* Output Layer (Binary Classification)

---

## 📂 Dataset

* MRI Brain Scan Images
* Preprocessed (Resized, Normalized)
* Train/Test Split

## ⚙️ Installation & Setup

```bash

```

### Install Dependencies

```bash

```

### Run Application

```bash
python app.py
```
